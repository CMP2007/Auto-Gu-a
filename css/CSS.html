Estilos en elementos HTML

Por si mismos los elementos HTML no se enfocan en brindarle una imagen de interzas muy agradable a los usuarios, su enfoque unicamente es el de definir la estructura y los elementos que se inplementaran en la paguina, para los estilos la herramienta que se utiliza es el CSS, esta tecnologuía esta enfocada unicamente en enriqueser la visualización de la paguina, esto lo hace atravez de un archivo CSS en el cual se definen todas las caracteristicas con las que se mostraran los elementos HTML, para esto cada grupos de caracteres debe de vincularse con los respectivos elementos en los que estos se aplicaran, esto lo hace atraves de tres formas de vinculación:

  • relacionandose atravez del tipo de elemento
  • relacionandose atraves del atributo "id" del elemento
  • relacionandose atraves de una clase, la cual es llamada por el elemento HTML con el atributo "class" el cual puede ser inplementado en cualquier elemento HTML
